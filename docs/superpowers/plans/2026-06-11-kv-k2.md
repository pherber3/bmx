### Task 1: `src/bmx/cache/codecs.py` + tests

**Files:** Create `src/bmx/cache/codecs.py`, `tests/test_cache_codecs.py`.

Public API:

```python
CACHE_ARMS = ("rtn_token", "rtn_channel", "rotate_rtn_token",
              "turboquant_mse", "turboquant_prod", "lowrank_rtn_channel")

def quantize_cache(arm: str, M: torch.Tensor, *, bits: int, seed: int = 0,
                   group: int = 64, rank: int = 0) -> tuple[torch.Tensor, float]:
    """M: (S, C) fp32 token-by-channel cache matrix. Returns
    (M_hat, bits_per_entry) where bits_per_entry counts ALL metadata."""
```

Arm semantics + bit accounting (b = bits, S×C matrix):
1. `rtn_token` — groupwise symmetric RTN along the channel dim per token (reuse `bmx.quant.rtn.rtn_quantize` directly on (S, C); group divides C). bpe = b + 16/group.
2. `rtn_channel` — KIVI-style: symmetric RTN along the TOKEN dim per channel = `rtn_quantize(M.mT, b, group).mT` with group dividing S (pad-free: assert S % group == 0). bpe = b + 16/group.
3. `rotate_rtn_token` — QuaRot-style: rotate channels (Hadamard if C is a power of 2 via `randomized_hadamard`, else `random_orthogonal`; seed-generated, 0 bits) then `rtn_token`, rotate back. bpe = b + 16/group.
4. `turboquant_mse` — per token: store ||x|| (fp16), rotate the normalized vector, quantize each coordinate with a precomputed b-bit Gaussian Lloyd-Max codebook scaled by 1/√C, unrotate, rescale. bpe = b + 16/C.
   Codebook helper (module-level, cached): `gaussian_codebook(bits, n_samples=2**18, n_iter=40, seed=0)` — 1-D Lloyd on a fixed seeded N(0,1) sample; deterministic; assign via bucketize on midpoints.
5. `turboquant_prod` — two-stage: `turboquant_mse` at (b−1) bits, then 1-bit QJL on the residual: with seed-generated S_qjl ∈ R^{C×C} (Gaussian), residual r reconstructed as r̂ = ||r||·√(π/2)/C · S_qjlᵀ sign(S_qjl r̂_unit) per token (the unbiased-IP linear reconstruction). bpe = (b−1) + 1 + 32/C (two fp16 norms). Requires b ≥ 2.
6. `lowrank_rtn_channel` — the K1-margin arm: L = `truncated_svd(M, rank)` (from `bmx.decomp.lrs`), residual R = M − L quantized with `rtn_channel`, M_hat = L + R̂. bpe = b + 16/group + 16·rank·(S+C)/(S·C). rank must be > 0 for this arm; other arms ignore rank.

Tests (offline, seeded; fp32):
- bit accounting: exact bpe formulas for representative (S, C, b, group, rank).
- monotonicity: for each arm, rel error at b=4 < rel error at b=2 on a seeded Gaussian matrix.
- determinism: same seed → identical M_hat (rotate/turboquant/QJL arms); different seed → different.
- `rtn_channel` beats `rtn_token` on a planted rogue-channel matrix (one column ×50 norm) at equal bits — the structural sanity that motivates KIVI.
- `gaussian_codebook(b)` beats uniform RTN on N(0,1) data in MSE at b=3 (Lloyd-Max optimality sanity); codebook is sorted, length 2^b.
- `turboquant_prod` unbiasedness: for fixed x, y and 64 seeds, mean of <y, M_hat-row> within 3 std-errors of <y, x> (the QJL stage's defining property).
- `lowrank_rtn_channel` at full rank ≈ exact (residual ~0); unknown arm raises.

Steps: failing tests → implement → pass → full gate → `git add src/bmx/cache/codecs.py tests/test_cache_codecs.py` (NO commit).
