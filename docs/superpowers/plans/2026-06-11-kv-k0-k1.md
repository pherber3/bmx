### Task 1: `src/bmx/cache/collect.py` + offline tests

**Files:** Create `src/bmx/cache/__init__.py` (empty), `src/bmx/cache/collect.py`, `tests/test_cache_collect.py`.

Public API (exact signatures):

```python
def collect_cache(model, input_ids: torch.Tensor, n_q_keep: int = 256) -> dict[str, torch.Tensor]:
    """Single prefill forward; returns {f"layer{i}.k": (h_kv, S, d), f"layer{i}.v": (h_kv, S, d),
    f"layer{i}.q": (h, n_q_keep, d), f"layer{i}.k_pre": (h_kv, S, d)} in fp16.

    k/v come from past_key_values (post-RoPE for RoPE models). q and k_pre come
    from forward hooks on the per-layer q/k projection modules, reshaped to heads
    and (for q) truncated to the LAST n_q_keep positions. Works for GPT2LMHeadModel
    (c_attn split; no RoPE so k_pre == k) and Llama-family (q_proj/k_proj hooks;
    k_pre is pre-RoPE). Batch dim must be 1."""

def save_cache(tensors: dict, path: str | Path) -> None: ...   # safetensors.torch.save_file
def load_cache(path: str | Path) -> dict[str, torch.Tensor]: ...
```

Implementation notes: detect family via `model.config.model_type` ("gpt2" vs "llama"/"qwen2"/"qwen3" — anything with `q_proj`). Head counts from config (`num_key_value_heads` fallback `num_attention_heads`; GPT-2: `n_head`). GPT-2 c_attn output is (S, 3d) → split → reshape (S, h, d) → permute (h, S, d). Llama hooks capture module outputs of shape (1, S, h*d) → reshape likewise. Run under `torch.no_grad()`, `use_cache=True`; remove hooks in `finally`.

Tests (offline, tiny random models — GPT2Config(n_layer=2, n_head=2, n_embd=32, vocab_size=97, n_positions=64) and LlamaConfig(num_hidden_layers=2, num_attention_heads=4, num_key_value_heads=2, hidden_size=32, intermediate_size=64, vocab_size=97, max_position_embeddings=64)):
1. Shapes/keys: all four tensors per layer, correct (h, S, d) shapes for both families, q has n_q_keep (= min(n_q_keep, S)) positions.
2. **GPT-2 invariant: `k_pre` ≈ `k`** (no RoPE; atol 1e-2 in fp16).
3. **Llama invariant: `k_pre` ≠ `k` but per-vector norms match** (RoPE is norm-preserving per head vector; rel diff < 1e-2).
4. save/load round-trip equality.

Steps: write failing tests → run (`uv run pytest tests/test_cache_collect.py -v`, expect ImportError) → implement → tests pass → full gate → `git add src/bmx/cache tests/test_cache_collect.py` (NO commit).

### Task 2: `src/bmx/cache/metrics.py` + tests

**Files:** Create `src/bmx/cache/metrics.py`, `tests/test_cache_metrics.py`.

```python
def logit_distortion(K: torch.Tensor, Kq: torch.Tensor, Q: torch.Tensor) -> float:
    """Mean over heads of ||Q K_q^T - Q K^T||_F / ||Q K^T||_F.
    K, Kq: (h_kv, S, d); Q: (h, T, d). GQA: repeat_interleave K to h heads."""

def attn_output_distortion(K, V, Kq, Vq, Q) -> float:
    """Same shapes; rel Frobenius error of softmax(Q K^T / sqrt(d)) V, mean over heads.
    Pass Kq=K or Vq=V to isolate one side."""
```

Tests: identity (Kq=K → 0.0); scaling K by (1+1e-3) gives logit_distortion ≈ 1e-3 (within 1e-5); GQA path (h=4, h_kv=2) runs and equals the manually-expanded computation; attn_output_distortion(K,V,K,V,Q) == 0 and is finite/positive under noise.

Steps: failing tests → implement → pass → full gate → `git add` (NO commit).

### Task 3: K1 census experiment + GPT-2 run

**Files:** Modify `experiments/frontier_breakeven.py` (one-line refactor: give `matrix_row` an optional `model_label: str = ""` parameter used instead of `cfg.model` — keep `cfg: Config` arg but only read `max_side_bpw` and the label; preserve existing callers/tests). Print path + total MB.

`experiments/k1_cache_census.py`: tyro Config(cache_path: str, model_label: str = ""). For each layer and each of {k, k_pre, v}: build the (S, h·d) matrix; record per row:
- break-even margins via the refactored `matrix_row` (this is E1's main number),
- per-channel outlier_mass (channels = h·d coords over tokens): mean/max,
- kurtosis (dim=-1 over channels per token AND over tokens per channel — record both),
- kurtosis after `randomized_hadamard` rotation over the channel dim (pad/truncate NOT allowed — if h·d not a power of 2, use `random_orthogonal` instead; say which was used in a column),
- channel-norm max/median ratio (the rogue-channel inventory, as in the 70B layer-0 verification).
Plus the same for q (shape (h·T, d)? No — q matrix as (T, h·d), same convention). Write parquet via artifacts (`create_run("k1_cache_census", cfg)`), per-row prints like d1.

Run: `uv run python experiments/collect_cache.py --model-name gpt2 --seq-len 1024` then `uv run python experiments/k1_cache_census.py --cache-path results/cache/gpt2_1024.safetensors --model-label gpt2`. Report the headline numbers (margins, rogue ratios, rotation dose-response) in your task report. Cache .safetensors files: do NOT stage (they're raw data, not metrics — add `results/cache/` to .gitignore). Stage code + the census run dir.
